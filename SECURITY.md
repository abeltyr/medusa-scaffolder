## Reporting a Vulnerability

We take the security of this project seriously. If you have any concerns or have found a vulnerability, please do not file a public issue.

Instead, please send a report to the maintainers privately according to the method below:

1. **Email**: Send details to the repository owner's email address (GitHub profile contact).
2. **Subject**: Please include "Security Vulnerability Report" in the subject line.
3. **Content**: Include as much detail as possible:
   - Type of vulnerability
   - Steps to reproduce
   - Affected components
   - Potential impact

We will:
- Acknowledge your report within 48 hours.
- Investigate the issue and determine its severity.
- Keep you informed of our progress.
- Publicly announce the fix once it is released (crediting you if desired).

Thank you for helping keep the community safe!
