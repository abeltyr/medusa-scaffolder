# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-03-20

### Added
- Initial release of Medusa Scaffolder CLI
- Support for generating:
  - TypeScript Interfaces (HTTP, Modules, Query, Service)
  - Workflows (Create, Update, Delete, Unlink)
  - Service Layer (CRUD operations)
  - Middleware (Admin, Store)
- Intelligent model parsing using `ts-morph`
- Modular generation architecture
- Comprehensive README and documentation
